Modelo OSI

Modelo OSI quando falamos sobre o modelo osi podemos dizer que o modelo osi trata-se de um modelo de referência para interconexão de sistemas abertos, desenvolvido pela Organização Internacional para Padronização I.S.O..

O modelo osi tem como objetio principal permitir a troca de informações entre computadores de diferêntes fabricantes usando uma arquitetura única. Este modelo esta dividido em 7 camadas.

O modelo osi fornece o desenvolvimento coordenado de padrões para a interconexão de redes:

  • Não especificando os protocolos de cada camada;
  • Apenas indíca as funções de cada camada;
  • Existem diferêntes opções de serviços e protocolos padronizados para as camadas;
  • Compatitibilidade obtida somente quando os sistemas optam pelos mesmos serviços e protocolos para todas as camadas.

Principais Funções das Camadas

Camada Física

  • Trata tensões e impulsos eletricos;
  • Especifica conectores e interfaces;
  • Providencia o fluxo de bits atraves do meio de transmissão.

Camada de Enlase

  • Deteta e opcionalmente corrigi erros;
  • Divide a cadeia de bits em quadros;
  • Delimita e reconhece quadros.
  • Realiza controle de fluxo;

Camada de rede

  • Identifica os endereços dos sistemas na rede;
  • Permite a interconexão de redes;
  • Controle de congestionamento;
  • Estabelece a rota mais adequada.
